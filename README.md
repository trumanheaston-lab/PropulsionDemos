# Propulsion Demos

Personal collection of interactive tools and hardware projects focused on rocket propulsion systems.  
Built while in high school with one clear goal: **gain real engineering experience and eventually design propulsion systems at SpaceX, Rocket Lab, or similar companies.**

### What’s Inside

**Software Demos**
- Rocket Equation Calculator – interactive tool showing how specific impulse, delta-v, and mass ratio actually affect propellant needs
- More demos coming (delta-v mission planners, Isp vs thrust trade-offs, nozzle performance)

**Hardware Projects**
- TVC Mount (Thrust Vector Control Gimbal) – 3D-printed two-axis servo gimbal for model rocket motors  
  → CAD files, build photos, test videos, and control code

### Why These Projects

These demos are designed to help me deeply understand the fundamental physics and engineering trade-offs that real propulsion teams deal with every day:
- How the rocket equation drives vehicle design decisions
- Why Isp, thrust, and mass matter differently for launch vehicles vs. in-space propulsion
- Mechanical challenges of gimballing engines for attitude control

The orbital mechanics and SSA platform I built with my team (VectraSpace) lives separately — this repo is focused purely on propulsion.

### Goal

Prove that I can:
- Build useful technical tools from scratch
- Design and fabricate physical hardware
- Think like an engineer about real propulsion problems

Open to feedback and always learning. If you're working in propulsion, I'd love to hear what demos or projects would be most useful to add next.
